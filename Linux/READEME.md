# Linux Personal Lab

## Introduction
Thos project documents my hands-on-practice with Linux using a virtual machine. The goal is to become familliar with basic command-line operations used in IT support 
and cybersecurity.

## Commands Practiced
- 'ls' - show current directory
- 'cat' - view file content
- 'nano' - edit file
- 'echo' - prints text to the terminal

## Network Commands
- 'ping' - used to test connectivity to external servers
- 'ip a' - used to view IP address and network interface details

## Activities
- Navigated directories using CLI
- Created and managed files and folders
- Edited a text file using 'nano'
- Viewed the file content using 'cat'
- Practiced basic terminal commands
- Tested internet connectivity using 'ping google.com'
- Verified system IP address using 'ip a'
